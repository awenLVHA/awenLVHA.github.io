gitpage+hexo

背景

想要在github上创建一个人博客，怎么搞～gitpage来帮忙，hexo来助攻！

搭建gitpage

STEP1 创建仓库

如图

STEP2 输入自己的项目名字，后面一定要加.github.io后缀，README初始化也要勾上。

STEP3 然后项目就建成了，点击Settings，向下拉到最后有个GitHub Pages，点击Choose a theme选择一个主题。然后等一会儿，再回到GitHub Pages，会变成下面这样：

STEP4 点击蓝色链接，就能访问啦～～

—华丽的分割线—–
gitpage已经搭建好了，如果你还希望将博客进一步优化，扩充，使用已有模版，可以使用hexo

安装hexo

STEP1 安装git bash（如已安装可跳忽略）

STEP2 cd到你的博客的文件夹，执行npm i hexo-cli -g

STEP3 文件夹根目录：hexo init以及 npm install

STEP4 执行:hexo s预览：localhost:4000
